## Project Description

**Cricket Auction Web Application**

This project is a web-based Cricket Auction system developed using **HTML, CSS, and JavaScript**. The application is designed to manage a cricket league auction where multiple team owners can bid for players, track budgets, and complete purchases in a simple and interactive interface. The system provides a full-screen auction dashboard that allows users to conduct player bidding in real time with automatic balance calculation and purchase tracking.

The application starts with a landing page that displays the league name, number of owners, and number of players per team. From the landing page, the user can open the auction screen where the main bidding process takes place. The auction page shows all owners with their available balance and maximum allowed bid. Each player row allows owners to enter the player name, place a bid, and confirm the purchase.

The system automatically controls the auction rules. Each owner has a fixed budget, and the program calculates the maximum bid allowed based on the remaining balance and remaining players to be purchased. The minimum bid amount is fixed, and the system prevents invalid bids such as exceeding balance or exceeding the maximum allowed bid. After a player is purchased, the owner’s balance is reduced automatically, and the purchase is recorded.

The interface is designed using CSS styling with responsive layout so that the auction table, owner boxes, and player rows are clearly visible. JavaScript is used to handle bidding logic, balance calculation, purchase validation, and screen updates without reloading the page.

The project is useful for conducting local cricket tournaments, college leagues, or friendly matches where player auctions are required. Instead of managing the auction manually, this system provides a digital solution that is fast, accurate, and easy to use.

The main objective of this project is to demonstrate how frontend web technologies can be used to build an interactive real-time auction system with proper rules, validation, and user-friendly design.
